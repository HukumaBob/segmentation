Project: Creating an Annotation Dataset for Endoscopic Images and Training a Neural Network

Project Overview

The goal of this project is to create an annotated dataset for images step-by-step based on endoscopic procedures, such as:

  • Gastroscopy
  • Colonoscopy
  • Bronchoscopy
  • Endoscopic Ultrasound (EUS)

The annotated dataset will be used to train a neural network for automatic detection of:

  1. Anatomical landmarks
  2. Pathological findings
  3. Quality assessment of the procedure

This project supports both offline mode (post-processing) and online mode (real-time analysis).

Image Segmentation is powered by SAM2.
Training and Detection are performed using YOLO v11.

Project Structure

The project consists of the following steps:

  • Image Annotation: Use an interface to create and extrapolate masks across image sequences.
  • Mask Generation and Extrapolation: Masks are manually created for a single frame and then extrapolated to the rest of the frames.
  • Tags and Mask Management: Masks are linked to anatomical landmarks or pathologies and stored in the database.
  • Data Preparation for Model Training: The final dataset is used to train a neural network.
Key Features
Users select a specific frame and create a mask. Masks are saved in the database and displayed on the annotated images.
An extrapolation model automatically applies annotations across all frames.
Masks can be toggled for visibility, deleted, or managed within the sequence.
Installation and Setup

To set up the project:

  1. Install dependencies using pip install -r requirements.txt.
  2. Configure the database and run migrations.
  3. Run the local server with python manage.py runserver.
Technologies Used

The project uses the following technologies:

  • SAM2: For image segmentation (Read more).
  • YOLO v11: For object detection and training.
